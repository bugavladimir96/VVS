21  /***
22   * An {@link Escaper} that converts literal text into a format safe for
23   * inclusion in a particular context (such as an XML document). Typically (but
24   * not always), the inverse process of "unescaping" the text is performed
25   * automatically by the relevant parser.
26   *
27   * <p>For example, an XML escaper would convert the literal string {@code
28   * "Foo<Bar>"} into {@code "Foo&lt;Bar&gt;"} to prevent {@code "<Bar>"} from
29   * being confused with an XML tag. When the resulting XML document is parsed,
30   * the parser API will return this text as the original literal string {@code
31   * "Foo<Bar>"}.
32   *
33   * <p><b>Note:</b> This class is similar to {@link CharEscaper} but with one
34   * very important difference. A CharEscaper can only process Java
35   * <a href="http://en.wikipedia.org/wiki/UTF-16">UTF16</a> characters in
36   * isolation and may not cope when it encounters surrogate pairs. This class
37   * facilitates the correct escaping of all Unicode characters.
38   *
39   * <p>As there are important reasons, including potential security issues, to
40   * handle Unicode correctly if you are considering implementing a new escaper
41   * you should favor using UnicodeEscaper wherever possible.
42   *
43   * <p>A {@code UnicodeEscaper} instance is required to be stateless, and safe
44   * when used concurrently by multiple threads.
45   *
46   * <p>Several popular escapers are defined as constants in the class {@link
47   * CharEscapers}. To create your own escapers extend this class and implement
48   * the {@link #escape(int)} method.
49   *
50   *
51   */
52  public abstract class UnicodeEscaper implements Escaper {
53      /*** The amount of padding (chars) to use when growing the escape buffer. */
54      private static final int DEST_PAD = 32;
55  
56      /***
57       * Returns the escaped form of the given Unicode code point, or {@code null}
58       * if this code point does not need to be escaped. When called as part of an
59       * escaping operation, the given code point is guaranteed to be in the range
60       * {@code 0 <= cp <= Character#MAX_CODE_POINT}.
61       *
62       * <p>If an empty array is returned, this effectively strips the input
63       * character from the resulting text.
64       *
65       * <p>If the character does not need to be escaped, this method should return
66       * {@code null}, rather than an array containing the character representation
67       * of the code point. This enables the escaping algorithm to perform more
68       * efficiently.
69       *
70       * <p>If the implementation of this method cannot correctly handle a
71       * particular code point then it should either throw an appropriate runtime
72       * exception or return a suitable replacement character. It must never
73       * silently discard invalid input as this may constitute a security risk.
74       *
75       * @param cp the Unicode code point to escape if necessary
76       * @return the replacement characters, or {@code null} if no escaping was
77       *     needed
78       */
79      protected abstract char[] escape(int cp);
80  
81      /***
82       * Scans a sub-sequence of characters from a given {@link CharSequence},
83       * returning the index of the next character that requires escaping.
84       *
85       * <p><b>Note:</b> When implementing an escaper, it is a good idea to override
86       * this method for efficiency. The base class implementation determines
87       * successive Unicode code points and invokes {@link #escape(int)} for each of
88       * them. If the semantics of your escaper are such that code points in the
89       * supplementary range are either all escaped or all unescaped, this method
90       * can be implemented more efficiently using {@link CharSequence#charAt(int)}.
91       *
92       * <p>Note however that if your escaper does not escape characters in the
93       * supplementary range, you should either continue to validate the correctness
94       * of any surrogate characters encountered or provide a clear warning to users
95       * that your escaper does not validate its input.
96       *
97       * <p>See {@link PercentEscaper} for an example.
98       *
99       * @param csq a sequence of characters
100      * @param start the index of the first character to be scanned
101      * @param end the index immediately after the last character to be scanned
102      * @throws IllegalArgumentException if the scanned sub-sequence of {@code csq}
103      *     contains invalid surrogate pairs
104      */
105     protected int nextEscapeIndex(CharSequence csq, int start, int end) {
106         int index = start;
107         while (index < end) {
108             int cp = codePointAt(csq, index, end);
109             if (cp < 0 || escape(cp) != null) {
110                 break;
111             }
112             index += Character.isSupplementaryCodePoint(cp) ? 2 : 1;
113         }
114         return index;
115     }

117     /***
118      * Returns the escaped form of a given literal string.
119      *
120      * <p>If you are escaping input in arbitrary successive chunks, then it is not
121      * generally safe to use this method. If an input string ends with an
122      * unmatched high surrogate character, then this method will throw
123      * {@link IllegalArgumentException}. You should either ensure your input is
124      * valid <a href="http://en.wikipedia.org/wiki/UTF-16">UTF-16</a> before
125      * calling this method or use an escaped {@link Appendable} (as returned by
126      * {@link #escape(Appendable)}) which can cope with arbitrarily split input.
127      *
128      * <p><b>Note:</b> When implementing an escaper it is a good idea to override
129      * this method for efficiency by inlining the implementation of
130      * {@link #nextEscapeIndex(CharSequence, int, int)} directly. Doing this for
131      * {@link PercentEscaper} more than doubled the performance for unescaped
132      * strings (as measured by {@link CharEscapersBenchmark}).
133      *
134      * @param string the literal string to be escaped
135      * @return the escaped form of {@code string}
136      * @throws NullPointerException if {@code string} is null
137      * @throws IllegalArgumentException if invalid surrogate characters are
138      *         encountered
139      */
140     public String escape(String string) {
141         int end = string.length();
142         int index = nextEscapeIndex(string, 0, end);
143         return index == end ? string : escapeSlow(string, index);
144     }

146     /***
147      * Returns the escaped form of a given literal string, starting at the given
148      * index.  This method is called by the {@link #escape(String)} method when it
149      * discovers that escaping is required.  It is protected to allow subclasses
150      * to override the fastpath escaping function to inline their escaping test.
151      * See {@link CharEscaperBuilder} for an example usage.
152      *
153      * <p>This method is not reentrant and may only be invoked by the top level
154      * {@link #escape(String)} method.
155      *
156      * @param s the literal string to be escaped
157      * @param index the index to start escaping from
158      * @return the escaped form of {@code string}
159      * @throws NullPointerException if {@code string} is null
160      * @throws IllegalArgumentException if invalid surrogate characters are
161      *         encountered
162      */
163     protected final String escapeSlow(String s, int index) {
164         int end = s.length();
165 
166         // Get a destination buffer and setup some loop variables.
167         char[] dest = DEST_TL.get();
168         int destIndex = 0;
169         int unescapedChunkStart = 0;
170 
171         while (index < end) {
172             int cp = codePointAt(s, index, end);
173             if (cp < 0) {
174                 throw new IllegalArgumentException(
175                                                    "Trailing high surrogate at end of input");
176             }
177             char[] escaped = escape(cp);
178             if (escaped != null) {
179                 int charsSkipped = index - unescapedChunkStart;
180 
181                 // This is the size needed to add the replacement, not the full
182                 // size needed by the string.  We only regrow when we absolutely must.
183                 int sizeNeeded = destIndex + charsSkipped + escaped.length;
184                 if (dest.length < sizeNeeded) {
185                     int destLength = sizeNeeded + (end - index) + DEST_PAD;
186                     dest = growBuffer(dest, destIndex, destLength);
187                 }
188                 // If we have skipped any characters, we need to copy them now.
189                 if (charsSkipped > 0) {
190                     s.getChars(unescapedChunkStart, index, dest, destIndex);
191                     destIndex += charsSkipped;
192                 }
193                 if (escaped.length > 0) {
194                     System.arraycopy(escaped, 0, dest, destIndex, escaped.length);
195                     destIndex += escaped.length;
196                 }
197             }
198             unescapedChunkStart
199                 = index + (Character.isSupplementaryCodePoint(cp) ? 2 : 1);
200             index = nextEscapeIndex(s, unescapedChunkStart, end);
201         }
202 
203         // Process trailing unescaped characters - no need to account for escaped
204         // length or padding the allocation.
205         int charsSkipped = end - unescapedChunkStart;
206         if (charsSkipped > 0) {
207             int endIndex = destIndex + charsSkipped;
208             if (dest.length < endIndex) {
209                 dest = growBuffer(dest, destIndex, endIndex);
210             }
211             s.getChars(unescapedChunkStart, end, dest, destIndex);
212             destIndex = endIndex;
213         }
214         return new String(dest, 0, destIndex);
215     }

217     /***
218      * Returns an {@code Appendable} instance which automatically escapes all
219      * text appended to it before passing the resulting text to an underlying
220      * {@code Appendable}.
221      *
222      * <p>Unlike {@link #escape(String)} it is permitted to append arbitrarily
223      * split input to this Appendable, including input that is split over a
224      * surrogate pair. In this case the pending high surrogate character will not
225      * be processed until the corresponding low surrogate is appended. This means
226      * that a trailing high surrogate character at the end of the input cannot be
227      * detected and will be silently ignored. This is unavoidable since the
228      * Appendable interface has no {@code close()} method, and it is impossible to
229      * determine when the last characters have been appended.
230      *
231      * <p>The methods of the returned object will propagate any exceptions thrown
232      * by the underlying {@code Appendable}.
233      *
234      * <p>For well formed <a href="http://en.wikipedia.org/wiki/UTF-16">UTF-16</a>
235      * the escaping behavior is identical to that of {@link #escape(String)} and
236      * the following code is equivalent to (but much slower than)
237      * {@code escaper.escape(string)}: <pre>{@code
238      *
239      *   StringBuilder sb = new StringBuilder();
240      *   escaper.escape(sb).append(string);
241      *   return sb.toString();}</pre>
242      *
243      * @param out the underlying {@code Appendable} to append escaped output to
244      * @return an {@code Appendable} which passes text to {@code out} after
245      *     escaping it
246      * @throws NullPointerException if {@code out} is null
247      * @throws IllegalArgumentException if invalid surrogate characters are
248      *         encountered
249      *
250      */
251     public Appendable escape(final Appendable out) {
252         assert out != null;
253 
254         return new Appendable() {
255             int pendingHighSurrogate = -1;
256             char[] decodedChars = new char[2];
257 
258             public Appendable append(CharSequence csq) throws IOException {
259                 return append(csq, 0, csq.length());
260             }
261 
262             public Appendable append(CharSequence csq, int start, int end)
263                 throws IOException {
264                 int index = start;
265                 if (index < end) {
266                     // This is a little subtle: index must never reference the middle of a
267                     // surrogate pair but unescapedChunkStart can. The first time we enter
268                     // the loop below it is possible that index != unescapedChunkStart.
269                     int unescapedChunkStart = index;
270                     if (pendingHighSurrogate != -1) {
271                         // Our last append operation ended halfway through a surrogate pair
272                         // so we have to do some extra work first.
273                         char c = csq.charAt(index++);
274                         if (!Character.isLowSurrogate(c)) {
275                             throw new IllegalArgumentException(
276                                                                "Expected low surrogate character but got " + c);
277                         }
278                         char[] escaped =
279                             escape(Character.toCodePoint((char) pendingHighSurrogate, c));
280                         if (escaped != null) {
281                             // Emit the escaped character and adjust unescapedChunkStart to
282                             // skip the low surrogate we have consumed.
283                             outputChars(escaped, escaped.length);
284                             unescapedChunkStart += 1;
285                         } else {
286                             // Emit pending high surrogate (unescaped) but do not modify
287                             // unescapedChunkStart as we must still emit the low surrogate.
288                             out.append((char) pendingHighSurrogate);
289                         }
290                         pendingHighSurrogate = -1;
291                     }
292                     while (true) {
293                         // Find and append the next subsequence of unescaped characters.
294                         index = nextEscapeIndex(csq, index, end);
295                         if (index > unescapedChunkStart) {
296                             out.append(csq, unescapedChunkStart, index);
297                         }
298                         if (index == end) {
299                             break;
300                         }
301                         // If we are not finished, calculate the next code point.
302                         int cp = codePointAt(csq, index, end);
303                         if (cp < 0) {
304                             // Our sequence ended half way through a surrogate pair so just
305                             // record the state and exit.
306                             pendingHighSurrogate = -cp;
307                             break;
308                         }
309                         // Escape the code point and output the characters.
310                         char[] escaped = escape(cp);
311                         if (escaped != null) {
312                             outputChars(escaped, escaped.length);
313                         } else {
314                             // This shouldn't really happen if nextEscapeIndex is correct but
315                             // we should cope with false positives.
316                             int len = Character.toChars(cp, decodedChars, 0);
317                             outputChars(decodedChars, len);
318                         }
319                         // Update our index past the escaped character and continue.
320                         index += (Character.isSupplementaryCodePoint(cp) ? 2 : 1);
321                         unescapedChunkStart = index;
322                     }
323                 }
324                 return this;
325             }
326 
327             public Appendable append(char c) throws IOException {
328                 if (pendingHighSurrogate != -1) {
329                     // Our last append operation ended halfway through a surrogate pair
330                     // so we have to do some extra work first.
331                     if (!Character.isLowSurrogate(c)) {
332                         throw new IllegalArgumentException(
333                                                            "Expected low surrogate character but got '" + c +
334                                                            "' with value " + (int) c);
335                     }
336                     char[] escaped =
337                         escape(Character.toCodePoint((char) pendingHighSurrogate, c));
338                     if (escaped != null) {
339                         outputChars(escaped, escaped.length);
340                     } else {
341                         out.append((char) pendingHighSurrogate);
342                         out.append(c);
343                     }
344                     pendingHighSurrogate = -1;
345                 } else if (Character.isHighSurrogate(c)) {
346                     // This is the start of a (split) surrogate pair.
347                     pendingHighSurrogate = c;
348                 } else {
349                     if (Character.isLowSurrogate(c)) {
350                         throw new IllegalArgumentException(
351                                                            "Unexpected low surrogate character '" + c +
352                                                            "' with value " + (int) c);
353                     }
354                     // This is a normal (non surrogate) char.
355                     char[] escaped = escape(c);
356                     if (escaped != null) {
357                         outputChars(escaped, escaped.length);
358                     } else {
359                         out.append(c);
360                     }
361                 }
362                 return this;
363             }
364 
365             private void outputChars(char[] chars, int len) throws IOException {
366                 for (int n = 0; n < len; n++) {
367                     out.append(chars[n]);
368                 }
369             }
370         };
371     }

373     /***
374      * Returns the Unicode code point of the character at the given index.
375      *
376      * <p>Unlike {@link Character#codePointAt(CharSequence, int)} or
377      * {@link String#codePointAt(int)} this method will never fail silently when
378      * encountering an invalid surrogate pair.
379      *
380      * <p>The behaviour of this method is as follows:
381      * <ol>
382      * <li>If {@code index >= end}, {@link IndexOutOfBoundsException} is thrown.
383      * <li><b>If the character at the specified index is not a surrogate, it is
384      *     returned.</b>
385      * <li>If the first character was a high surrogate value, then an attempt is
386      *     made to read the next character.
387      *     <ol>
388      *     <li><b>If the end of the sequence was reached, the negated value of
389      *         the trailing high surrogate is returned.</b>
390      *     <li><b>If the next character was a valid low surrogate, the code point
391      *         value of the high/low surrogate pair is returned.</b>
392      *     <li>If the next character was not a low surrogate value, then
393      *         {@link IllegalArgumentException} is thrown.
394      *     </ol>
395      * <li>If the first character was a low surrogate value,
396      *     {@link IllegalArgumentException} is thrown.
397      * </ol>
398      *
399      * @param seq the sequence of characters from which to decode the code point
400      * @param index the index of the first character to decode
401      * @param end the index beyond the last valid character to decode
402      * @return the Unicode code point for the given index or the negated value of
403      *         the trailing high surrogate character at the end of the sequence
404      */
405     protected static final int codePointAt(CharSequence seq, int index, int end) {
406         if (index < end) {
407             char c1 = seq.charAt(index++);
408             if (c1 < Character.MIN_HIGH_SURROGATE ||
409                 c1 > Character.MAX_LOW_SURROGATE) {
410                 // Fast path (first test is probably all we need to do)
411                 return c1;
412             } else if (c1 <= Character.MAX_HIGH_SURROGATE) {
413                 // If the high surrogate was the last character, return its inverse
414                 if (index == end) {
415                     return -c1;
416                 }
417                 // Otherwise look for the low surrogate following it
418                 char c2 = seq.charAt(index);
419                 if (Character.isLowSurrogate(c2)) {
420                     return Character.toCodePoint(c1, c2);
421                 }
422                 throw new IllegalArgumentException(
423                                                    "Expected low surrogate but got char '" + c2 +
424                                                    "' with value " + (int) c2 + " at index " + index);
425             } else {
426                 throw new IllegalArgumentException(
427                                                    "Unexpected low surrogate character '" + c1 +
428                                                    "' with value " + (int) c1 + " at index " + (index - 1));
429             }
430         }
431         throw new IndexOutOfBoundsException("Index exceeds specified range");
432     }

434     /***
435      * Helper method to grow the character buffer as needed, this only happens
436      * once in a while so it's ok if it's in a method call.  If the index passed
437      * in is 0 then no copying will be done.
438      */
439     private static final char[] growBuffer(char[] dest, int index, int size) {
440         char[] copy = new char[size];
441         if (index > 0) {
442             System.arraycopy(dest, 0, copy, 0, index);
443         }
444         return copy;
445     }
